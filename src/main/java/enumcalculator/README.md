### Enum을 사용해 문자열 계산기 만들기

구현해야 할 기능
- [x] decideOperation : 입력받은 연산자가 어떤 계산을 하는 연산자인지 구분해줌.
- [x] calculate : 입력받은 두 수를 계산해줌
- [X] result : 입력받은 연산식을 계산한 결과를 반환해줌

예외 처리
- [x] case: DIVIDE의 두번째 숫자가 0이 되면 에러
